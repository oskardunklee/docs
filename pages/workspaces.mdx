A **workspace** is an isolation boundary for all data.

- Identifier: `workspaceId` (string). Default is your email domain (e.g., `spare.com`).
- All API reads require `workspaceId`. Many list endpoints accept `?workspaceId=<id>`.
- Uniqueness: `Client` has a composite unique key `{ workspaceId, domain }`.
